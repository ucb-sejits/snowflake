class stencil.vector.Vector¶

A Vector that supports arithmetic. These can be used for numpy indicies and has neighborhood utility functions.

classmethod moore_vectors(ndim, radius=1, closed=False)¶
Parameters:
  • ndim – Number of dimensions
  • radius – Moore Neighborhood radius
  • closed – Should the neighborhood include all points r < radius or only r = radius
Returns:

Iterator of vectors in the Moore neighborhood, \(\{v | v \in Z^n, ||v||_1 = r\}\)
classmethod von_neumann_vectors(ndim, radius=1, closed=False)¶
Parameters:
  • ndim – Number of dimensions
  • radius – Von Neumann Neighborhood radius
  • closed – Should the neighborhood include all points r < radius or only r = radius
Returns:

Iterator of vectors in the Von Neumann neighborhood, \(\{v | v \in Z^n, ||v||_\infty = r\}\)
